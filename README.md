# 프로젝트 주제 수정
 # [21.01.07]
 영화예매사이트에 데이터분석에 대하여 내용추가가 어렵다고 판단,
 코로나19사태로 인하여 집에서 iptv 및 영화다시보기에 사례가 많아지고 있으므로,
 다양하고 재미있는 영화를 보기 위하여, 본인의 선호도에 따른 영화 추천기능 및 리뷰를 통하여 
 다른사람들에게 영화를 추천할 수 있는 '영화 리뷰'사이트로 프로젝트 수정 결정
